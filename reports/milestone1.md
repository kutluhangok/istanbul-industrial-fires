# Milestone 1 Submission

## Current Status

As of Milestone 1, the project structure, research question, and data acquisition strategy have been established.

The project aims to analyze temporal, spatial, and environmental patterns in industrial fire incidents in Istanbul. The intended analysis pipeline includes:

* data collection,
* exploratory data analysis (EDA),
* hypothesis testing,
* and, if feasible, machine learning and spatial modeling.

## Data Collection Status

The main research data for this project depends on institutional fire incident records that are not yet publicly available in the required format.

At this stage:

* a formal academic support letter has been requested through my project advisor,
* institutional outreach for data access is in progress,
* and a meeting with GOSB Fire Department is planned for next week.

Following earlier guidance, I am also evaluating additional institutional sources such as reinsurance / insurance-related datasets and aggregated public sources.
